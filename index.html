Ronera Central Agustín Rodríguez Mena

Funcionalidad:

La aplicación de gestión integralpara la ronera combina innovación y eficiencia en su núcleo. Ofrece un sistema de gestión de inventario avanzado, capaz de predecir necesidades de reabastecimiento mediante análisis de patrones de consumo históricos, asegurando un stock óptimo.

Además, la aplicación automatiza y optimiza procesos de producción, desde la recepción de materias primas hasta el empaque final, generando informes detallados para identificar mejoras continuas. Un módulo de control de calidad riguroso garantiza la consistencia y calidad del ron, realizando pruebas automáticas y registros analizados para mantener altos estándares de producto.

Tecnologías utilizadas:
  • Java
  • IntelliJ IDEA
  • JavaFX
  • Scene Builder

Tareas realizadas:
  • Gestión de Inventario Avanzada: Implementé un sistema de gestión de inventario que no sólo registra y actualiza el stock de rones, sino que también integra un análisis predictivo para anticipar futuras necesidades de reabastecimiento.
  • Gestión de Procesos Eficiente: Diseñé e implementé un módulo de gestión de procesos que automatiza y optimiza varias etapas del proceso de producción del ron.
  • Control de Calidad Riguroso: Integré un sistema de control de calidad que monitoriza cada etapa del proceso de producción para garantizar la consistencia y la calidad del ron.

Web Quieres ser mi novia?.

Funcionalidad:

Una página web simple que, al interactuar con los botones, muestra mensajes gracioso y no deja rechazar la propuesta "¿Quieres ser mi novia?", comportándose de distintas maneras. Utilizamos HTML para la estructura, CSS para el estilo y JavaScript para la interactividad. Este enfoque sigue las mejores prácticas de desarrollo web, manteniendo el código limpio y fácil de entender.
